# INYATSI Department Files (Mobile)

React Native mobile dashboard with:

- Branded header (`INYATSI`, bookmark icon, search, bell, avatar)
- Breadcrumb (`Home > Departments`)
- Welcome section
- 2-column department grid (Engineering, Finance, HR, Procurement)
- Quick Access recent files list
- 5-tab bottom navigation with prominent Upload button

## Run

1. Install dependencies:

```bash
npm install
```

2. Start Metro:

```bash
npm start
```

3. Run app:

```bash
npm run android
```

or

```bash
npm run ios
```
